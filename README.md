# Task 02 - Customer Segmentation using K-Means Clustering

## Objective
Create a K-Means clustering algorithm to group customers of a retail store based on their purchase behavior.

## Dataset
Mall Customers Dataset

## Features Used
- Annual Income (k$)
- Spending Score (1-100)

## Steps Performed
1. Loaded dataset using Pandas
2. Selected important features
3. Applied Elbow Method to determine optimal clusters
4. Trained K-Means clustering model
5. Visualized customer segments and centroids

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Result
Successfully segmented customers into 5 clusters based on spending habits and annual income.
